<h1>Reto de desarrollo de un formulario multipasos de FrontEnd Mentor</h1>
<h2>Descripción del proyecto</h2>
El proyecto es una <b>solución al desafío “Multi-step form” de Frontend Mentor</b>, es decir, <b>un formulario web de varios pasos (wizard)</b> donde el usuario va avanzando paso a paso para completar información y elegir opciones antes de enviar o confirmar. 
Específicamente:
<ul>
  <li><b>Paso 1 – Your Info:</b> Pide al usuario sus datos personales (nombre, email, teléfono).</li>
  <li><b>Paso 2 – Select Plan:</b> Permite seleccionar un plan (por ejemplo “Arcade”, “Advanced” o “Pro”) y escoger facturación mensual o anual.</li>
  <li><b>Paso 3 – Add-ons:</b> Añadir complementos/servicios extra que modifican la experiencia.</li>
  <li><b>Paso 4 – Summary:</b> Muestra un resumen de las elecciones antes de finalizar.</li>
  <li><b>Confirmación:</b> Finalmente muestra una pantalla de agradecimiento tras confirmar.</li>
</ul>
El objetivo es practicar <b>interacciones JavaScript, validaciones simples, manejo de pasos</b>, y construir una experiencia responsiva e intuitiva.
<h2>Tecnologías utilizadas</h2>
<ul>
  <li><b>HTML</b> — para la estructura del formulario y pasos.</li>
  <li><b>CSS</b> — para estilos, layout y responsividad.</li>
  <li><b>JavaScript</b> — para manejar la navegación entre pasos, lógica de selección y validación del formulario.</li>
</ul>
<h2>Instrucciones de instalación y uso</h2>
<ul>
  <li>Para usar este código hay que ir a la pestaña Code | Download ZIP.</li>
  <li>Descomprimir el archivo zipeado, entrar a la carpeta creada y buscar el archivo <b>index.html</b>.</li>
  <li>Una vez localizado el archivo <b>index.html</b> le damos doble click a dicho archivo y se nos abrirá el proyecto HTML en nuestro navegador de Internet predeterminado.</li>
  <li>Una vez abierto en el navegador:</li>
  <ul>
    <li>Llena los campos del paso 1.</li>
    <li>Aquí en el paso 1 sino se pone el nombre seguido del apellido paterno y no se pone una dirección de correo electrónico valida y no se pone un número telefónico de por lo menos 10 dígitos no se podrá continuar al paso número 2.</li>
    <li>Haz clic en <b>Next Step</b> para avanzar.</li>
    <li>Elige plan y la frecuencia del mismo; es decir que si el plan deseado lo elegirás anual o mensual.</li>
    <li>Elige los complementos deseados.</li>
    <li>Visualiza el resumen.</li>
    <li>Da click en cambiar o en confirmar.</li>
    <li>Una vez que le diste click en confirmar veras la pantalla de agradecimiento.</li>
  </ul>
</ul>
